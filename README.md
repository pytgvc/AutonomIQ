🤖AutonomIQ is an AI-powered personal helper built using Streamlit and Google Gemini AI.

Live Application
Access the deployed application here:
🔗https://autonomiq-hkbkrjsux5gk2lnmq9uvku.streamlit.app/
Features

- Intelligent task analysis using Gemini AI
- Reminder management
- Notification system
- Email task processing
- Calendar event handling
- Alarm scheduling
- NumPy analytics dashboard

Tech Stack

- Python
- Streamlit
- Google Gemini API
- NumPy
- Schedule
- SMTP
- Plyer
- Python Dotenv

Installation

1. Clone the repository
2. Install dependencies:

pip install -r requirements.txt

3. Create a ".env" file:

GOOGLE_API_KEY=YOUR_API_KEY

4. Run the application:

streamlit run app.py
